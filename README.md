# K-Band Recommender

A simple scaffold for a Korean band recommendation web application using Streamlit for the frontend and FastAPI for the backend.

## 기술 스택

- Streamlit
- FastAPI
- Docker
- docker-compose

## 실행 방법

1. `docker-compose up --build`
2. `http://localhost` 에서 프론트엔드 접속
3. `http://localhost:8000` 에서 백엔드 API 접속
